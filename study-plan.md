# AI Agent Study Plan (Personal Roadmap)

Author: Haomin Feng

Goal: Learn AI Agent systems from fundamentals and build a
production-style AI Agent platform.

Estimated timeline: 6--9 months (10--12 hours/week)

Stages: 1. Foundations (LLM + Embeddings) 2. Retrieval-Augmented
Generation (RAG) 3. Tool-Using Agents 4. Memory + MCP + Multi-Agent
Systems 5. Capstone AI Agent Platform

## Stage 1 --- LLM & Embedding Foundations

Duration: 3--4 weeks

Learn: - Transformer basics - Tokenization - Embeddings - Vector
search - Vector databases

Resources: https://jalammar.github.io/illustrated-transformer/
https://stanford-cs324.github.io/winter2022/
https://platform.openai.com/docs/guides/embeddings
https://www.sbert.net/ https://docs.trychroma.com/

Project: Build a semantic search engine using: Python,
sentence-transformers, ChromaDB, FastAPI

## Stage 2 --- Retrieval Augmented Generation (RAG)

Duration: 4--6 weeks

Learn: - Document chunking - Hybrid retrieval - Reranking - RAG
evaluation

Resources: https://docs.llamaindex.ai/
https://python.langchain.com/docs/tutorials/rag/
https://arxiv.org/abs/2312.10997 https://www.trulens.org/

Project: Build a Personal Knowledge Assistant capable of answering
questions over your documents.

## Stage 3 --- Tool-Using AI Agents

Duration: 4--6 weeks

Learn: - ReAct pattern - Tool integration - Agent frameworks

Resources: https://arxiv.org/abs/2210.03629
https://github.com/langchain-ai/langchain
https://github.com/microsoft/semantic-kernel
https://github.com/microsoft/autogen https://github.com/crewAIInc/crewAI

Project: Create a Multi-Tool AI Assistant that can use APIs such as
weather, stock prices, and web search.

## Stage 4 --- Memory, MCP & Multi-Agent Systems

Duration: 6--8 weeks

Learn: - Agent memory systems - Model Context Protocol (MCP) -
Multi-agent collaboration

Resources: https://python.langchain.com/docs/modules/memory/
https://modelcontextprotocol.io/ https://github.com/modelcontextprotocol
https://github.com/microsoft/autogen

Project: Build an Autonomous Research Agent capable of planning,
researching, and summarizing information.

## Stage 5 --- Capstone Project

Duration: 6--10 weeks

Build a production-style AI Agent Platform.

Architecture: React Frontend Spring Boot Backend Python Agent Service
LLM + Tools + Vector Database

Capstone Idea: AI Developer Assistant

Capabilities: - analyze repositories - suggest bug fixes - generate
documentation - assist with deployment

## Weekly Study Schedule

3 hours reading 3 hours coding 2 hours experimentation 2 hours project
development

Total: about 10 hours per week

## Final Skillset

After finishing this roadmap you should understand:

-   LLM systems
-   embeddings
-   vector databases
-   RAG architecture
-   AI agents
-   tool integration
-   MCP
-   multi-agent systems
-   AI system architecture
